Algerian Forest Fires Prediction

Machine Learning model trained on Algerian_forest_fires_cleaned_dataset.csv

Introduction

Forest fires are one of the most dangerous natural disasters in Algeria. This project uses machine learning to predict the FWI (Fire Weather Index) based on key meteorological and fire-related features.

Features Used

  • Temperature (°C)
  • RH (%) – Relative Humidity
  • Ws (km/h) – Wind Speed
  • Rain (mm)
  • FFMC – Fine Fuel Moisture Code
  • DMC – Duff Moisture Code
  • DC – Drought Code
  • ISI – Initial Spread Index
  • BUI – Buildup Index
  • Classes (0,1) – Fire Occurrence
  • Region (0,1) – Bejaia / Sidi-Bel Abbes

Prediction Target

The model predicts the FWI (Fire Weather Index), which represents the potential risk and severity of a forest fire under given environmental conditions.
